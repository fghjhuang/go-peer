## 云端
云端提供2个功能，1个是桥接代理，就是转发服务端的数据到客户端，目前是做到一个服务里面，
以后可以考虑分离转发服务器，然后转发服务器做集群；再往后可以让客户端直接到RTMP流媒体服务器拉流，
然后等p2p打洞成功后自动切换到p2p并关闭RTMP。
1个是信令转发，通过stun服务获取各自的对外端口，然后通过此信令服务传输并打洞，此服务通过websocket技术实现。
